## DIGITAL-CLOCK ⏰

A simple digital clock built with **HTML**, **JavaScript**, and **CSS** . This project display the current time and update every second.

---

## Features 🚀

1. Showa hours, minutes, and seconds
2. updates automatically in real-time
3. Clean and minimal design


---

## Technology 🛠️

- **HTML**
- **JavaScript**
- **CSS** (for styling)
